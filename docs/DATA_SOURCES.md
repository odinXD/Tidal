# 데이터 소스 및 정책 안내

K-Terminal은 다양한 무료 데이터 소스를 통합하여 제공합니다.

## 1. 한국 주식 시세 및 차트
* **소스**: 네이버 금융 내부 API (`api.stock.naver.com`, `api.finance.naver.com`)
* **비용/제한**: 무료, 비공식 엔드포인트
* **주의사항**:
  * 비공식 엔드포인트이므로 상업적 사용은 권장하지 않습니다.
  * 브라우저 CORS 정책으로 인해 로컬 `proxy-server`를 경유하여 데이터를 호출합니다.
  * 과도한 요청 시 네이버 측에서 IP를 차단할 수 있습니다.

## 2. 기업 공시 (DART)
* **소스**: 금융감독원 DART OpenAPI
* **비용/제한**: 무료, 일 10,000건 제한
* **용도**: 종목별 공시 목록, 주요사항보고서, 기업 기본 정보, 재무제표 

## 3. 거시경제 매크로 지표
* **소스**: 한국은행 경제통계시스템 (ECOS) OpenAPI
* **비용/제한**: 무료, 일 1,000건 제한
* **용도**: 한국은행 기준금리, 시중 금리, 통화량 등 통계 조회

## 4. 글로벌 지수 및 환율
* **소스**: Frankfurter API (환율), Yahoo Finance (원자재/지수)
* **비용/제한**: 무료 (Frankfurter는 무제한, Yahoo는 비공식)
* **용도**: 실시간 원/달러 환율 종가 및 금, 유가 등 매크로 지표 보조용

## 상태 배지 정책
화면 하단 상태바(`StatusBar`)와 각 위젯 상단에 데이터의 출처와 상태를 표시합니다.
- 🟢 **실시간/정상**: 프록시를 통한 네이버 금융 시세 등
- 🟡 **지연**: 무료 API 특성상 15~20분 지연된 글로벌 데이터
- 🔴 **오류/키 없음**: DART, ECOS 등 API 키가 누락되었거나 한도를 초과한 경우
- **데이터 없음**: 가짜 더미 데이터를 화면에 렌더링하지 않고, 명확하게 "데이터 없음"을 표시합니다.
